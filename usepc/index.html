使っているPCを紹介します。

メインPC

mouse DAIV 4N (Laptop)
CPU 11th Intel Core i7 1165G7
GPU Intel Iris Xe G7 Graphics + Geforce GTX 1650Ti
メモリー DDR4-3200 8GBx2
SSD PCIe3.0x4接続 NVMe SSD 512GB(内部的にはPCIe4.0対応)
OS Windows 11 Home
ちょっと前(2021/12/5)に新入りしたパソコンです。来てからほとんど経っていないので詳しい評価はできませんが、ゲームをする際にパワーが出ているのを感じます。
初期搭載はWindows 10でしたが、Windows 11にアップグレードしました。Windows 10の時点でなぜかブルスクが多かったしShiftキーが割れてたりとなんかおかしいので初期不良の交換品の可能性ある気がしてます。

MacBook Pro (Laptop)
CPU Apple M1
GPU Apple M1
メモリー 16GBユニファイドメモリ
SSD 512GB

これは2021年2月から12月までの１０ヶ月間メインだったMacbook Proです。

Macはいいですね。デザインもかっこいいしおすすめです。まあWindowsのほうがいいけどね。
